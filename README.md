## Features
- Basic Shapes (Box, Sphere, Plane)
- Basic Lighting (Ambient, Directional, Spotlight)
- Textures (Background, Box)
- Importing Custom 3D Models
- Basic GUI Set Up
- Mouse Position and Ray Caster Events
- Basic Animations

## Installation & Running
- bash
- npm install
- npm run dev
